Proyectos personales

Goal Tracker - .Net + React

GoalTracker es una herramienta de gestión de objetivos y proyectos que te ayuda a organizar tu trabajo, visualizar tu progreso y mantener el control de tus deadlines. Categoriza proyectos, asigna tareas, y monitorea el estado de cada iniciativa en tiempo real.

.NET PostgreSQL CQRS + MediatR React + Tailwind CSS GitHub Actions
www.goaltracker.es

Harmonizer - .Net + Win Forms a migrar a MAUI

Harmonizer es una herramienta para poder analizar y descubrir armonías. Está en proceso de migración a MAUI para ser multiplataforma, pero actualmente es una aplicación de escritorio con Win Forms. Permite cargar archivos MIDI, analizar su contenido armónico y generar visualizaciones interactivas para descubrir nuevas armonías y progresiones. Se encuntra en desarrollo, pero ya cuenta con funcionalidades de análisis armónico, generación de progresiones y visualización de acordes. Es un proyecto personal que refleja mi pasión por la música y la tecnología, combinando mis habilidades técnicas con mi creatividad musical para crear una herramienta única para músicos y compositores.

.NET SqlLite CQRS + MediatR Win Form => MAUI GitHub Actions
github.com/arielgrillo/harmonizer

Backery Admin - Python + Django + React

Backery Admin es una aplicación para la gestión administrativa de una panadería. Permite controlar inventarios y ventas de manera eficiente. Está desarrollada con Python + Django en el backend y React en el frontend, utilizando Tailwind CSS para el diseño. La aplicación facilita la organización y optimización de las operaciones diarias, mejorando la productividad y el control del negocio.

Python + Django REST PostgreSQL CQRS + MediatR React + Tailwind CSS GitHub Actions
www.goaltracker.es
